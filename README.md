## Descripción del Proyecto

**AutoElite** es una landing page ficticia desarrollada como proyecto final universitario
para la materia de Sistemas Operativos. Simula el sitio oficial de una agencia automotriz
premium ubicada en Cuernavaca, Morelos, con catálogo de vehículos, sección de marcas
representadas, planes de financiamiento y testimonios de clientes.

El objetivo académico fue aplicar y demostrar competencias en maquetación web responsiva,
diseño UI/UX, manejo del DOM con JavaScript vanilla y uso de frameworks CSS.

---

## Tecnologías Utilizadas

| Tecnología           | Uso                                    |
| -------------------- | -------------------------------------- |
| HTML5 semántico      | Estructura y contenido                 |
| CSS3 + Variables CSS | Estilos, paleta de colores, responsive |
| Bootstrap 5          | Grid, navbar y componentes base        |
| Bootstrap Icons      | Iconografía                            |
| JavaScript Vanilla   | Slider automático, navbar dinámica     |

---

## Características del Sitio

- **Navbar fija** con ocultamiento inteligente al hacer scroll
- **Hero section** con fondo de imagen y overlay
- **Barra de estadísticas** animada con métricas de la agencia
- **Slider de catálogo** automático (8 s) con 6 vehículos
- **Sección de marcas** con 8 alianzas automotrices
- **Sección de asesoría** con fondo de patrón punteado
- **3 planes de financiamiento** con card destacada
- **6 testimonios** de clientes con sistema de estrellas
- **Footer completo** con contacto, navegación y redes sociales
- Diseño 100 % responsivo (mobile-first)
- Paleta elegante: negro profundo, dorado, crema marfil y acero
